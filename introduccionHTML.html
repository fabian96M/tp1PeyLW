INTRODUCCION HTML

Este es un curso que pretende introducir al lector en el mundo del HTML. No se requieren conocimientos previos. Se empezara desde cero. No hay que saber nada de programacion para utilizar HTML, basta con aprender un par de cosas sencillas para empezar a crear paginas.


Es MUY FACIL aprender HTML. Tan solo necesitas un editor de texto, un curso de html y un navegador web (como netscape o internet explorer), ni siquiera necesitas estar conectado a internet.
